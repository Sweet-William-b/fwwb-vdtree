System Positioning

不是展示屏，而是校园安防事件的工作入口。

鼎新智眼把视频监控、算法分析、人工确认和报告归档放在同一条链路里。首页负责建立可信的系统形象，后台控制台负责完成真实任务流：接入、分析、复核、导出。

Core Capability

从视频接入到事件闭环，保持轻量但完整。

01

实时预警

接入样例视频、本地文件或预留视频流入口，聚合异常片段和风险状态。

02

智能分析

结合行为识别、场景分段和风险映射，输出可复核的结构化事件。

03

留档追溯

支持人工修订、导出 JSON / CSV / ZIP，便于比赛展示和后续审计。

Response Workflow

把安防处置拆成可追踪的四步。

  1. 01 接入视频源

    选择样例、本地上传或预留 RTSP 流，为不同演示场景保留入口。

  2. 02 生成预警任务

    后端返回任务状态、事件数量、FPS、延迟和最近告警。

  3. 03 人工复核修订

    管理员确认风险等级、行为标签、误报状态和备注说明。

  4. 04 导出归档报告

    输出结构化 JSON、CSV、异常片段清单和 ZIP 资料包。
